汉方个人信息保护政策：

一、<strong>个人信息的定义</strong>
<br/>
个人信息是指以电子或者其他方式记录的能够单独或者与其他信息结合识别特定自然人身份或者反映特定自然人活动情况的各种信息。
<br/>

二、<strong>法律依据</strong>
<br/>
我们根据《中华人民共和国网络安全法》《中华人民共和国个人信息保护法》和《信息安全技术个人信息安全规范》（GB/T
35273-2020）以及其它相关法律法规的要求，并严格遵循正当、合法、必要的原则，出于您访问我们的平台、使用我们提供的服务和/或产品等过程中而收集和使用您的个人信息。
<br/>

三、<strong>用途变更</strong>
<br/>

如在后续服务过程中，我们需将您的个人信息用于本隐私政策未载明的其它用途，或基于特定的服务或目的将收集而来的信息用于其他目的，我们将以合理的方式向您告知并再次征得您的同意。
<br/>

四、<strong>匿名化处理</strong>
<br/>

在收集您的个人信息后，如我们需将您的个人信息用于本隐私政策未载明的用途，我们将通过技术手段对数据进行去匿名化处理，匿名化处理的信息将无法识别特定的主体。特别提醒您注意，我们将保证前述信息无法单独或结合其他信息识别到您的个人真实身份，其不属于法律意义上您的个人信息。请您了解并同意，在不透露您个人信息的前提下，您授权我们对用户数据库进行群体画像，数据分析并用于营销和为您提供检测结果服务建议。
<br/>

五、<strong>统计信息共享</strong>
<br/>

我们会对我们的产品与/或服务使用情况进行统计，并可能会与公众或第三方共享这些统计信息，以展示我们的产品与/或服务的整体使用趋势。但这些统计信息将进行第（四）条所述的匿名化处理，不包含您的任何身份识别信息。

我们如何收集和使用您的个人信息
<br/>

一、帮助您成为我们的在线注册用户
<br/>

为接受汉方舌诊小程序全面的服务，您应首先注册一个用户账号。 您同意通过微信向我们提供您的手机号为您注册并登录账号。
如您拒绝提供手机号码进行实名验证，将可能导致您无法注册并使用我们的产品及服务。

二、提供舌象健康自测服务

<br/>
汉方舌诊小程序向您提供舌象健康自测服务。为向您提供舌象检测服务，我们需获取您的相机权限以拍摄舌象照片。如您拒绝提供，将无法使用舌象健康自测服务，但是不影响您使用我们提供的其他服务。

根据您拍摄的舌象图片，我们会分析您的舌色、齿痕、点刺、裂痕、苔质、苔色，向您反馈您的舌象特征，利用已有的舌象诊断资料数据库作出舌象综合分析，向您反馈您的舌象解读，包括对您的舌色、点刺、裂纹、苔色、苔质、齿痕的单项评估结果及舌象总体评价结果，并向您提供舌象健康解读以及个性化健康养生建议，包括饮食建议、中医保健建议。
请您注意，拍摄光线及角度等因素可能影响舌象照片呈现结果，并进一步影响舌象评估结果。
如果您拒绝提供，我们将不能为您提供舌象检测服务，但不影响您使用我们的其他服务。

<br/>

当您选择通过微信第三方平台登陆使用我们的产品和/或服务时，我们可能从第三方处获取您授权共享的账户信息（头像、昵称）；同时您可能通过我们的产品和/或服务使用由我们的关联方、合作方提供的服务，您同意我们根据实际第三方平台的授权要求以及业务合作需求及情况，从该等第三方以及我们的关联方或合作方处接收、使用、汇总、分析经您授权同意其向我们提供的您的个人信息。

<br/>

您知晓，根据相关法律法规的规定，在以下情形中，我们可以在不征得您的知情同意的情况下而收集、使用相关个人信息：

五、共享、转让与公开披露
<br/>

共享

1. 我们不会在未取得您同意的情况下，向汉方以外的任何其他公司、组织和个人共享您的个人信息，但以下情况除外：

- 我们可能会根据法律法规规定，或按政府主管部门的强制性要求，对外共享您的个人信息。
- 只有共享您的信息，才能实现我们的产品与/或服务的核心功能或提供您需要的服务（此种情形我们将在情形发生时专门提示您并取得您的同意）。
- 与我们的附属公司共享：您授权汉方，将您提供给汉方的信息、享受汉方服务产生的信息，经匿名化处理后用于汉方、汉方关联公司为您提供服务、推荐产品、开展市场调查与信息数据分析。我们只会共享必要的个人信息，且受本隐私政策中所声明目的的约束。附属公司如要改变个人信息的处理目的，将再次征求您的授权同意。

2.与授权合作伙伴共享：仅为实现本政策中声明的目的，我们的某些服务将由授权合作伙伴提供。我们可能会与合作伙伴共享您的某些个人信息，以提供更好的客户服务和用户体验。例如，在您购买我们的产品时，我们必须与物流服务提供商共享您的个人信息才能安排送货，或者安排合作伙伴提供服务。我们仅会出于合法、正当、必要、特定、明确的目的共享您的个人信息，并且只会共享提供服务所必要的个人信息。我们的合作伙伴无权将共享的个人信息用于任何其他用途。如果您拒绝我们的合作伙伴在提供服务时收集为提供服务所必须的个人信息，将可能导致您无法在汉方舌诊小程序使用该第三方服务。

任何公司、组织和个人，但以下情况除外：

公开披露

我们仅会在以下情况下，公开披露您的个人信息：

1. 获得您明确同意后；
2. 基于法律的披露：在法律、法律程序、诉讼或政府主管部门强制性要求的情况下，我们可能会公开披露您的个人信息；
3. 在符合法律法规的前提下，当我们收到上述披露信息的请求时，我们会要求必须出具与之相应的法律文件，如传票或调查函。我们坚信，对于要求我们提供的信息，应该在法律允许的范围内尽可能保持透明。

在以下情形中，共享、转让、公开披露您的个人信息无需事先征得您的授权同意：

1. 与国家安全、国防安全直接相关的；
2. 与公共安全、公共卫生、重大公共利益直接相关的；
3. 与犯罪侦查、起诉、审判和判决执行等直接相关的；

6. 从合法公开披露的信息中收集个人信息的，如合法的新闻报道、政府信息公开等渠道；
7. 根据个人信息主体要求签订和履行合同所必需的；
8. 用于维护所提供的产品或服务的安全稳定运行所必需的，例如发现、处置产品或服务的故障；
